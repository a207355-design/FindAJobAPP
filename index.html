SDG 1: No Poverty

Goal: End poverty in all its forms everywhere.

Why it matters: Unemployment and underemployment are major drivers of poverty, yet many job seekers struggle to find suitable opportunities due to information barriers in the job market. Providing an accessible platform connects job seekers with employers, empowering individuals to secure stable incomes and improve their livelihoods.

Lab Submissions

Lab 1: UI & Layouts

Reflection: Building the basic UI introduced me to the fundamentals of Jetpack Compose. I learned how to effectively use Rows, Columns, and Modifiers to create a structured and responsive layout from scratch.

Lab 2: Interaction & State

Reflection: Adding state management using `remember` and `mutableStateOf` brought the app to life. It was fascinating to see how the UI reacts dynamically to user interactions like button clicks.

Lab 3: Material Design

Reflection: Implementing Material 3 components significantly improved the app's aesthetics. I now understand the critical role of typography, cohesive color schemes, and standardized UI elements in user experience.

Lab 4: Navigation

Reflection: Setting up the NavHost to navigate between multiple screens tied individual components into a cohesive application. Managing routes and passing data between screens was a major technical milestone.

Lab 5: Room Database

Reflection: Integrating the Room database introduced me to local data persistence in Android. Writing DAOs and using Kotlin Coroutines to perform asynchronous database operations was challenging but highly rewarding.

Projects

Project 1: Find A Job App (Base)

Problem Statement: Job seekers often lack access to consolidated employment information. This base app establishes the core navigation and UI to bridge that gap.

Lessons Learned: Architecting a multi-screen application using ViewModels to share state was crucial. I learned how to maintain clean code structure while handling complex UI navigation flows.

Project 2: Find A Job App (Advanced Integration)

Problem Statement: To effectively combat poverty (SDG 1), a job platform must provide real-time opportunities, location-aware matching, and community support. This update integrates live APIs, GPS sensors, and cloud databases to make the platform truly functional.

Lessons Learned: Integrating Firebase Firestore for cloud sharing and Retrofit for REST APIs expanded my understanding of connected applications. Utilizing device hardware (GPS Location) elevated the project from a static interface to a dynamic, real-world utility.

Reflection & Learning Journey

My journey from Lab 1 to Project 2 has been incredibly transformative. Initially, I struggled with grasping the declarative nature of Jetpack Compose layouts. However, as the course progressed, I transitioned from building static screens to developing a fully connected, hardware-aware application. Working consistently on the "Find A Job" app under the SDG 1 theme not only improved my technical skills in APIs, databases, and state management, but also gave my coding practice a meaningful, real-world purpose.
